# AutomaticDoorLock #
Real-Time door locking mechanism

COMPONENT LIST:
  5 1kΩ Resistors
  3 2kΩ Resistors
  2 Arduino UNO R3 (or equivalent)
  2 HC-05 Bluetooth Modules
  1 RFID-RC522 Module
  2 HC-SR04 Ultrasonic Sensor Modules
  1 High Torque Servo
  1 Magnetic Door Switch
  1 Green LED
  1 Red LED
  1 5V External Power Supply
  Jumper Wires

REQUIRED SOFTWARE:
  Arduino IDE

INSTRUCTIONS:
  Wire the Inside Unit Controller and Outside Unit Controller modules as shown in the diagrams below:
  [InsideUnitController_WiringDiagram.pdf](https://github.com/user-attachments/files/23942425/InsideUnitController_WiringDiagram.pdf)
  [OutsideUnitController_WiringDiagram.pdf](https://github.com/user-attachments/files/23942440/OutsideUnitController_WiringDiagram.pdf)
